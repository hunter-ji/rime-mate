# Rime Mate (Rime 配置助手)

**让 macOS 上的 Rime (鼠须管) 输入法配置变得简单、直观、可视化。**

![Rime Mate 截图](./rime-mate.jpg)

Rime 是一款极具可玩性的输入法，但其复杂的配置文件劝退了不少用户。**Rime Mate** 专为非程序员用户设计，你不再需要面对复杂的配置文件。通过简单的菜单选择，即可轻松完成词库安装、功能扩展和日常维护。

## ✨ 核心功能

- **可视化配置**: 告别代码编辑器，通过直观的菜单界面管理你的输入法。
- **扩展功能一键装**: 想要更强大的联想能力？一键安装语言模型等高级扩展，无需手动下载和配置。
- **词库轻松管理**: 无论是“雾凇拼音”还是“薄荷输入法”，都能轻松安装和更新。
- **零门槛上手**: 自动处理复杂的配置路径和依赖关系，你只需关注打字体验本身。

## 🚀 快速开始

### 1. 安装

打开你的“终端”应用 (Terminal)，复制并运行以下命令：

```bash
/bin/bash -c "$(curl -fsSL https://github.com/hunter-ji/rime-mate/releases/latest/download/setup.sh)"
```

脚本会自动下载最新版本，并在你的 Rime 配置目录中创建一个快捷方式。

### 2. 使用

安装完成后，你无需记忆任何命令：

1. 打开 Rime 配置文件夹（`~/Library/Rime`）。
2. 你会看到一个名为 **`Rime配置助手.command`** 的文件，**双击运行**它。
3. 在弹出的窗口中，使用键盘 `↑` `↓` 选择功能，按 `Enter` 确认即可。

### 3. 更新

当有新版本发布时，只需**再次运行上述安装命令**即可。脚本会自动检测并更新到最新版本，无需手动删除旧版。

### 4. 卸载

在 Rime 的配置文件夹中，删除`Rime配置助手.command`文件和`rime-mate-config`文件夹。

## ❓ 常见问题

#### Q: 我需要懂编程才能用吗？
A: 完全不需要。本工具就是为了让不懂编程的用户也能轻松配置 Rime 而开发的。

#### Q: 它安全吗？
A: 本项目完全开源，所有代码均公开可见。它只会修改 Rime 的配置文件，不会上传你的任何隐私数据。
